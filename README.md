# AI-Based Object Detection System

Overview

This project is a web application that detects objects in images using the YOLOv8 deep learning model. Users can upload an image, and the system will identify objects and display them with bounding boxes.

Technologies Used

* Python
* YOLOv8 (Ultralytics)
* Streamlit
* NumPy
* Pillow

Features

* Upload an image
* Detect objects using AI
* Display detected objects with bounding boxes
